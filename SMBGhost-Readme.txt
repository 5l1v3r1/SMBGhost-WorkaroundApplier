SMBGhost Workarounds Scanner for CVE-2020-0796 - SMBv3 unauthenticated RCE

This scanner has the ability to detect and apply the workaround in case your system is not patched yet.
The Scanner is looking for the vulrability in your system only.
You should run this script as administrator.

Libraries used in this project:

* winreg 
* platform, os
* os
* ctypes

Usage: python3 SMBGhost-workarounds

For more informaiton check:
https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/adv200005
